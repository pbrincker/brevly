Teste da Página de Erro 404

Instruções para teste:

  1. Certifique-se de que o servidor backend está rodando na porta 3333
  2. Certifique-se de que o frontend está rodando na porta 5173
  3. Clique nos links abaixo para testar diferentes cenários

Links de teste:

Teste 1: Link inexistente (deve redirecionar para 404)
Teste 2: Outro link inexistente
Teste 3: Acesso direto à página 404

O que verificar:

  • ✅ Os links inexistentes devem redirecionar para http://localhost:5173/404
  • ✅ A página 404 deve exibir o SVG com efeito glitch
  • ✅ A página deve ter o título "Link não encontrado"
  • ✅ A página deve ter a mensagem explicativa
  • ✅ A página deve ter o link "brev.ly"
  • ✅ A página deve ter o botão "Voltar ao início"
